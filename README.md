# Mini Etherscan

Live Dome [Mini Etherscan](https://mini-etherscan.vercel.app/)

Connecting to Ethereum mainnet and fetch data <br/>
Using Ethers.js for connection and metamask wallet

# Install Dep

```shell
npm install
```

# Libraries

<list>
<ul>Vite + React</ul>
<ul>Ethers.js</ul>
<ul>PicoCSS</ul>

# Run the Site

```bash
npm run dev
```
